# **App Name**: ClarityCode

## Core Features:

- Code Deobfuscation: Submit obfuscated code to the AI tool and receive a deobfuscated version.
- Functionality Explanation: Explanation of the original code's functionality (generated by AI).
- Code Editor: Simple code editor with syntax highlighting for input and output.
- Download Output: Option to download the deobfuscated code.

## Style Guidelines:

- Primary color: Dark blue (#283593) for a professional look.
- Secondary color: Light gray (#EEEEEE) for code editor background.
- Accent: Teal (#26A69A) for buttons and highlights.
- Monospace font for code display.
- Split-screen layout with input on the left and output on the right.
